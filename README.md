# Weather Balloon Telemetry

Development of telemetry, communication, and tracking systems for high-altitude weather balloon experimentation.

<br>

## Project Goals
- GPS tracking
- Telemetry communication
- Environmental sensing
- Payload design
- Data visualization

<br>

## Research Areas
- RF propagation
- Packet reliability
- Antenna performance
- Power optimization
- High-altitude telemetry

<br>

## Planned Hardware
- Arduino / ESP32
- GPS modules
- Radio transceivers
- Environmental sensors
- Battery systems

<br>

## Status
Project planning and architecture development.
